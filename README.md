###### Setup:

set twilio env variables
```
TWILIO_ACCOUNT_SID=xxxxxxxxxxxx
TWILIO_AUTH_TOKEN=xxxxxxxxxxxxx
```

(optional but recommended) create venv


install requirements
```
pip install -r requirements.txt
```


run
```
python3 ui.py
```
